Loop Design

How the agent works

1Thought

The model inspects the goal, memory, and scratchpad to decide what to do next.

2Action

If a tool is needed, the model emits a tool name plus JSON input instead of guessing.

3Observation

The tool result is fed back into the scratchpad and becomes evidence for the next reasoning step.

4Final Answer

Once the loop has enough evidence, the agent exits with a user-facing answer.
